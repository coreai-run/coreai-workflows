# Hello World Workflow

This is a minimal workflow template demonstrating the COREAI workflow structure.

## What it does

1. Greets the user
2. Shows current date/time
3. Displays active COREAI profile

## Usage

Just invoke this workflow and it will run automatically.

## Config

The greeting message can be customized via config.

!`coreai config show hello-world`
